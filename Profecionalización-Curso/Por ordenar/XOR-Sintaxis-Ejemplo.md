#### Sintaxis básica

```c#
expresión1 ^ expresión2
```
#### Ejemplo básico
```c#
bool tieneDescuento = true;
bool esMiembro = false;

if (tieneDescuento ^ esMiembro) {
    Console.WriteLine("Tienes un beneficio especial.");
}
```
